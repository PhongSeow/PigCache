# PigCache
#### [English Readme](https://github.com/PhongSeow/PigCache/blob/master/README.md)

PigCache 是一个缓存系统，有多种不同的应用场景。

## ***目录和文件描述***

### Release

发布执行码目录，如果你不想看到源程序，你可以直接使用这个目录中的文件。

##### Release\DotNet
类库的DLL文件和 NuGet 包。

#### Release\DotNet\PigCacheLib
最轻量的类库，是需要客户端。

### Src

源码目录。

#### Src\DotNet\PigCacheLib

类库目录
